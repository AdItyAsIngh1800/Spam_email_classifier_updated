# Spam_email_classifier_updated


## Overview
This project builds a machine learning model to classify SMS messages as spam or non-spam (ham).  
Dataset: [SMS Spam Collection](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset)

## Workflow
1. Exploratory Data Analysis (EDA)
2. Data Preprocessing (text cleaning, TF-IDF vectorization)
3. Model Training (Logistic Regression, Naive Bayes)
4. Evaluation (accuracy, precision, recall, F1)
5. Saving Model

## Results
| Model | Accuracy | Precision | Recall | F1 Score |
|--------|-----------|------------|----------|-----------|
| Naive Bayes | 97.1% | 0.96 | 0.95 | 0.96 |

## Technologies Used
- Python
- scikit-learn
- pandas, numpy
- matplotlib, seaborn

## How to Run
1. Clone the repository  
2. Install dependencies: `pip install -r requirements.txt`  
3. Run the notebook: `jupyter notebook spam_classifier.ipynb`
